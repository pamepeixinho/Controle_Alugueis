Projeto de Controle de Alugueis
========

Este projeto foi criado com o objetivo de facilitar o gereciamento de alugueis, provendo uma interface gráfica criada com QT usando a linguagem C++. 

Projeto criado juntamente com os desenvolvedores Victor Roveda e Giulia Ferri, o mesmo foi apresentado como projeto na matéria Técnicas de Programação I, utilizando conceitos de OOP.
